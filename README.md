# 🛡️ Azure Cloud VM Metadata Enumerator (Metasploit Module)

## 📖 Overview
[cite_start]This is a **Custom Metasploit Auxiliary Module** developed to perform post-exploitation reconnaissance on **Azure Cloud Infrastructure**[cite: 52]. [cite_start]The module specifically targets the **Azure Instance Metadata Service (IMDS)** to extract sensitive metadata about a Virtual Machine from an internal perspective [cite: 209-210, 292].

---

## ✨ Key Features (Information Gathered)
The module automates the extraction of the following critical data points:
* [cite_start]**VM Identification:** Virtual Machine Name & Unique Instance ID [cite: 170, 182-183].
* [cite_start]**Network Infrastructure:** Public & Private IP Addresses [cite: 171-172, 187-188].
* [cite_start]**Geographical Data:** Specific Azure Region/Location of the resource[cite: 177, 184].
* [cite_start]**System Specs:** VM Size (SKU) and Operating System details [cite: 185-186].
